# MyChat 安装与使用指南

> 本插件是 WoW 12.0.x 的聊天增强插件，遵循"增强而不接管"原则：只增强原生聊天，不替换聊天框，出问题可随时降级或关闭。
> 设计与开发细节见 [wow_chat_plugin_dev_tasks.md](./wow_chat_plugin_dev_tasks.md) 与 [wow_chat_plugin_execution_plan.md](./wow_chat_plugin_execution_plan.md)。

---

## 1. 安装

插件本体位于本目录的 `MyChat/` 文件夹（与 `MyChat.toc` 同级）。

### 1.1 找到魔兽的插件目录

把 `MyChat` 整个文件夹复制到对应版本的 `Interface\AddOns\` 下：

| 版本 | 目标路径 |
|---|---|
| 正式服（Retail） | `World of Warcraft\_retail_\Interface\AddOns\` |
| 测试服（PTR） | `World of Warcraft\_ptr_\Interface\AddOns\` |

- **Windows** 示例：`C:\Program Files (x86)\World of Warcraft\_retail_\Interface\AddOns\MyChat\`
- **macOS** 示例：`/Applications/World of Warcraft/_retail_/Interface/AddOns/MyChat/`

### 1.2 目录结构应如下

```text
Interface/AddOns/MyChat/
  MyChat.toc          <- 插件清单
  Init.lua
  Core.lua
  ...其余 .lua
  Data/Defaults.lua
  UI/SettingsPanel.lua
  UI/MirrorPanel.lua
  UI/ChannelBar.lua
```

> 关键：`MyChat.toc` 必须直接位于 `AddOns\MyChat\` 下，而不是多套一层文件夹（不要出现 `AddOns\MyChat\MyChat\MyChat.toc`）。

### 1.3 在游戏中启用

1. 启动游戏，进入**角色选择界面**。
2. 左下角点击 **「插件」（AddOns）** 按钮。
3. 勾选 **MyChat**。
4. 建议勾选左下角 **「加载过期插件」**（若显示版本不符时）。
5. 进入游戏。加载成功后聊天框会打印：
   `MyChat: v0.1.0 loaded. Type /mychat for options.`

### 1.4 版本说明

- 插件 `## Interface:` 标记为 `120007, 120005, 120000`（覆盖 12.0.7 / 12.0.5 / 12.0.0）。若游戏为 12.0.x 的其他小版本而提示"过期"，勾选「加载过期插件」即可正常使用；确认可用后可自行把 `.toc` 第一行改为你客户端的实际版本号（正式服可在游戏内用 `/dump (select(4, GetBuildInfo()))` 查看当前 Interface 号）。

## 2. 快速上手（30 秒）

1. 进入游戏，输入 `/mychat` 查看命令帮助，或 `/mychat config` 打开设置面板。
2. 默认已开启的增强：关键词与自己名字高亮、重复折叠、密语记录、自动加入「大脚世界频道」。
3. 想要频道快速切换条：在设置里勾选 **「启用频道切换条」**（默认关闭）。
4. 收到密语后，输入 `/mychat reply 你好` 即可快速回复最近一位密语对象。

---

## 3. 命令一览（`/mychat`）

| 命令 | 作用 |
|---|---|
| `/mychat` | 显示命令帮助（一定有文字输出，可用来确认插件在工作） |
| `/mychat config` | 打开设置面板 |
| `/mychat bar` | 开关频道切换条 |
| `/mychat debug` | 输出诊断信息（模块状态、错误数、最近消息、已加入频道等） |
| `/mychat reset` | 恢复默认设置（随后需 `/reload` 使全部生效） |
| `/mychat safe` | 开启安全模式（仅保留只读增强，关闭高风险功能） |
| `/mychat reply <内容>` | 回复最近一位密语对象 |

> 说明：`/mychat` 空命令现在打印帮助而非直接开面板，确保一定有可见反馈；开面板请用 `/mychat config`。
> `/mychat reply` 在战斗中会自动延迟到脱战后发送，避免战斗中触发发送限制。

## 4. 设置面板

打开方式：`/mychat config`，或 **游戏菜单 → 选项 → 插件（AddOns）→ MyChat**。

面板按分组组织，每一项都真实生效（不保留任何"装饰性"空开关；12.0 原生已做的事如时间戳、职业色不再重复提供）。

### 频道切换条
| 项 | 说明 | 默认 |
|---|---|---|
| 启用频道切换条 | 在聊天框下方显示频道快速切换按钮条 | 关 |
| 显示「说/队/团/会」按钮 | 分别控制四个基础按钮是否出现 | 开 |
| 频道按钮带编号前缀 | 按钮前加频道编号（如 `6大脚`），关则只显示短名（如 `大脚`） | 关 |
| 频道按钮用短名 | 截取频道名前几个字，关则显示完整名 | 开 |
| 短名取字符数 | 短名截取前几个字符（1–8，如设 2 则「大脚世界频道」→「大脚」） | 2 |
| 频道短名映射 | 每行 `频道名=显示名`，命中时优先于自动截取（如 `大脚世界频道=大脚`） | 空 |

> 社区/俱乐部频道（`GetChannelList` 返回 `Community:<clubId>:<streamId>` 这种内部标识）会自动通过 Blizzard 原生的 `ChatFrame_ResolveChannelName` 解析为社区名（与原生聊天、其它插件一致，如「漫天星辰」而非流名「综合」）；社区数据尚未加载时先显示占位「社区」，加载完成后自动刷新。若想固定成某个名字，也可在上面的「频道短名映射」里写 `Community:876127095:1=我的社区`。

### 高亮与关键词
| 项 | 说明 | 默认 |
|---|---|---|
| 高亮自己的名字 | 消息中出现自己名字时高亮（绿色） | 开 |
| 高亮关键词 | 逗号分隔的关键词列表，命中变金色（可在面板增删） | 来T, 来奶, 集合 |

### 重复折叠
| 项 | 说明 | 默认 |
|---|---|---|
| 折叠重复消息 | 短时间内重复刷屏折叠为 `(重复 xN)`，不删除 | 开 |
| 重复判定时间窗口 | 多少秒内的相同消息算重复（1–60） | 8 |

### 密语
| 项 | 说明 | 默认 |
|---|---|---|
| 记录最近密语对象 | 记录密语来往对象，供 `/mychat reply` 快捷回复 | 开 |
| 记住最近密语人数 | 保留最近多少个密语对象（1–50） | 10 |

### 自动加入频道
| 项 | 说明 | 默认 |
|---|---|---|
| 登录后自动加入 | 逗号分隔的频道名列表（可在面板增删） | 大脚世界频道 |

### 镜像面板
| 项 | 说明 | 默认 |
|---|---|---|
| 启用镜像面板 | 独立的镜像消息面板（不接管主聊天框） | 关 |
| 镜像系统消息 | 是否把系统消息也镜像进来（默认关，避免噪音刷屏） | 关 |
| 按频道分色显示 | 镜像行按频道类型着色 | 开 |
| 最多保留行数 | 镜像面板最多显示多少行（20–500） | 100 |

### 高级
| 项 | 说明 | 默认 |
|---|---|---|
| 安全模式 | 仅保留只读增强（高亮/折叠/镜像），**禁用密语代发**（自动加入频道不受影响，属低风险操作） | 关 |

> 大多数项改动后即时生效（复选框点击、数值/列表输入框按回车提交）。个别涉及已存在结构的改动可 `/reload` 兜底。

## 5. 功能使用说明

### 5.1 原生聊天增强
MyChat 遵循"增强而不替换"：**不重复 12.0 原生已经做好的事**（时间戳、频道名、玩家名职业色都由原生负责）。MyChat 只做原生没有的增强：关键词/自己名字高亮、重复折叠等。关插件后原生聊天完全不受影响。

### 5.2 关键词与自己名字高亮
- 默认关键词：`来T`、`来奶`、`集合`（命中变金色）。
- 消息中出现你的角色名时高亮（绿色）。
- 关键词可在 `/mychat config` → 「高亮与关键词」里直接增删（逗号分隔，回车提交）。

### 5.3 重复消息折叠
同一人在设定窗口（默认 8 秒，可在面板调整）内重复的相同消息会折叠显示为 `原文 (重复 xN)`，**只折叠不删除**。不同人发相同内容不会误折叠。

### 5.4 最近密语快捷回复
- 收到或发出密语后，对象会被记入"最近密语"列表（保留人数可在面板调整）。
- 用 `/mychat reply <内容>` 回复最近一位对象；无历史时会提示"无最近密语对象"。
- 开启安全模式时，`/mychat reply` 不会发送（安全模式禁用一切代发/代操作）。

### 5.5 自动加入频道
- 登录、切图或 `/reload` 后，会自动加入配置里的频道（默认 `大脚世界频道`，可在面板增删）。
- 加入采用延迟 + 有限次重试，避免登录瞬间失败或刷屏。
- 自动加入频道属低风险操作，**不受安全模式影响**（安全模式只禁用密语代发这类会替你操作游戏的功能）。

### 5.6 频道快速切换条（默认关闭）
1. 在设置里勾选 **「启用频道切换条」**。
2. 聊天框下方出现一排按钮：基础的 **说 / 队 / 团 / 会**（可分别开关），加上每个已加入编号频道各一个按钮。
3. 点击某个按钮 → 打开魔兽**原生聊天输入框**并切到对应频道，**光标就位，你打字后按回车发送**（插件不代发，零风险）。
4. 不在队伍/团队/公会时，对应按钮自动置灰。
5. 按钮条可拖动；战斗中开启/移动会延迟到脱战后执行。

### 5.7 镜像面板（默认关闭）
可选的独立消息面板，把消息按频道着色镜像显示，**不接管主聊天框**。在设置里勾选「启用镜像面板」开启，可拖动。

## 6. 常见问题（FAQ）

**Q：加载后没反应 / 聊天框没提示加载信息？**
- 确认目录为 `AddOns\MyChat\MyChat.toc`（没有多套一层文件夹）。
- 角色选择界面的插件列表里确认 MyChat 已勾选。
- 若列表里根本没有 MyChat，多半是路径放错了版本目录（`_retail_` vs `_ptr_`）。

**Q：插件显示"已过期"无法加载？**
- 在插件列表勾选「加载过期插件」；或把 `MyChat.toc` 首行 `## Interface:` 改为你客户端的实际版本号。

**Q：频道切换条不显示？**
- 默认是关闭的，需在 `/mychat` 设置里勾选「启用频道切换条」。

**Q：某个功能出问题 / 聊天异常？**
- 先 `/mychat safe` 进入安全模式（只保留只读增强）。
- 再 `/mychat debug` 查看诊断信息。
- 仍有问题可 `/mychat reset` 恢复默认设置后 `/reload`。
- 最后可在插件列表取消勾选 MyChat，原生聊天不受任何影响。

**Q：设置会保存吗？**
- 会。设置存于 `SavedVariables\MyChatDB`，`/reload` 或重登后保留。`/mychat reset` 只重置功能开关，不清除最近密语/已加入频道等运行时数据。

---

## 7. 卸载

- 临时停用：角色选择界面的插件列表取消勾选 MyChat。
- 彻底删除：删除 `Interface\AddOns\MyChat\` 文件夹。存档 `WTF\...\SavedVariables\MyChatDB.lua` 可一并删除以清除配置。
