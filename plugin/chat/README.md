# MyChat 安装与使用指南

> 本插件是 WoW 12.0.x 的聊天增强插件，遵循"增强而不接管"原则：只增强原生聊天，不替换聊天框，出问题可随时降级或关闭。
> 设计与开发细节见 [wow_chat_plugin_dev_tasks.md](./wow_chat_plugin_dev_tasks.md) 与 [wow_chat_plugin_execution_plan.md](./wow_chat_plugin_execution_plan.md)。

---

## 1. 安装

插件本体位于本目录的 `MyChat/` 文件夹（与 `MyChat.toc` 同级）。

### 1.1 找到魔兽的插件目录

把 `MyChat` 整个文件夹复制到对应版本的 `Interface\AddOns\` 下：

| 版本 | 目标路径 |
|---|---|
| 正式服（Retail） | `World of Warcraft\_retail_\Interface\AddOns\` |
| 测试服（PTR） | `World of Warcraft\_ptr_\Interface\AddOns\` |

- **Windows** 示例：`C:\Program Files (x86)\World of Warcraft\_retail_\Interface\AddOns\MyChat\`
- **macOS** 示例：`/Applications/World of Warcraft/_retail_/Interface/AddOns/MyChat/`

### 1.2 目录结构应如下

```text
Interface/AddOns/MyChat/
  MyChat.toc          <- 插件清单
  Init.lua
  Core.lua
  ...其余 .lua
  Data/Defaults.lua
  UI/SettingsPanel.lua
  UI/MirrorPanel.lua
  UI/ChannelBar.lua
```

> 关键：`MyChat.toc` 必须直接位于 `AddOns\MyChat\` 下，而不是多套一层文件夹（不要出现 `AddOns\MyChat\MyChat\MyChat.toc`）。

### 1.3 在游戏中启用

1. 启动游戏，进入**角色选择界面**。
2. 左下角点击 **「插件」（AddOns）** 按钮。
3. 勾选 **MyChat**。
4. 建议勾选左下角 **「加载过期插件」**（若显示版本不符时）。
5. 进入游戏。加载成功后聊天框会打印：
   `MyChat: v0.1.0 loaded. Type /mychat for options.`

### 1.4 版本说明

- 插件 `## Interface:` 标记为 `120000`（对应 12.0.0）。若游戏为 12.0.x 的其他小版本而提示"过期"，勾选「加载过期插件」即可正常使用；确认可用后可自行把 `.toc` 第一行改为你客户端的实际版本号。

## 2. 快速上手（30 秒）

1. 进入游戏，输入 `/mychat` 打开设置面板。
2. 默认已开启：时间戳、频道缩写、职业色、关键词与自己名字高亮、重复折叠、密语记录。
3. 想要频道快速切换条：在设置里勾选 **「启用频道切换条」**（默认关闭）。
4. 收到密语后，输入 `/mychat reply 你好` 即可快速回复最近一位密语对象。

---

## 3. 命令一览（`/mychat`）

| 命令 | 作用 |
|---|---|
| `/mychat` | 打开设置面板 |
| `/mychat debug` | 输出诊断信息（模块状态、错误数、最近消息、已加入频道等） |
| `/mychat reset` | 恢复默认设置（随后需 `/reload` 使全部生效） |
| `/mychat safe` | 开启安全模式（仅保留只读增强，关闭高风险功能） |
| `/mychat reply <内容>` | 回复最近一位密语对象 |

> 说明：`/mychat reply` 在战斗中会自动延迟到脱战后发送，避免战斗中触发发送限制。

## 4. 设置面板

打开方式：`/mychat`，或 **游戏菜单 → 选项 → 插件（AddOns）→ MyChat**。

面板内可切换以下开关（括号为默认值）：

| 开关 | 说明 | 默认 |
|---|---|---|
| 显示时间戳 | 每条消息前显示 `[时:分]` | 开 |
| 频道缩写 | 频道名显示为短标签（世界/交易/队/团/会…） | 开 |
| 玩家名职业色 | 按职业给玩家名着色 | 开 |
| 高亮自己的名字 | 消息中出现自己名字时高亮 | 开 |
| 记录最近密语 | 记录密语来往对象，供快捷回复 | 开 |
| 折叠重复消息 | 短时间内重复刷屏折叠为 `(重复 xN)`，不删除 | 开 |
| 世界/交易/队伍分流 | 给消息打逻辑视图标签（供镜像面板/统计） | 开 |
| 启用镜像面板 | 开启一个独立的镜像消息面板（不接管主聊天框） | 关 |
| 启用频道切换条 | 在聊天框上方显示频道快速切换按钮条 | 关 |
| 安全模式（仅只读增强） | 关闭高风险功能，只保留只读增强 | 开 |

> 修改开关后大多即时生效；涉及频道恢复等结构性设置在提示时 `/reload` 生效。

## 5. 功能使用说明

### 5.1 原生聊天增强（默认开启）
无需操作，登录后聊天框自动带上：时间戳、频道缩写、玩家名职业色。关插件后原生聊天完全不受影响。

### 5.2 关键词与自己名字高亮
- 默认关键词：`来T`、`来奶`、`集合`（命中变金色）。
- 消息中出现你的角色名时高亮（绿色）。
- 关键词列表目前在 `Data/Defaults.lua` 的 `highlightKeywords` 中，`/mychat reset` 会恢复默认。

### 5.3 重复消息折叠
同一人在约 8 秒窗口内重复的相同消息会折叠显示为 `原文 (重复 xN)`，**只折叠不删除**。不同人发相同内容不会误折叠。

### 5.4 最近密语快捷回复
- 收到或发出密语后，对象会被记入"最近密语"列表。
- 用 `/mychat reply <内容>` 回复最近一位对象；无历史时会提示"无最近密语对象"。

### 5.5 自动加入 / 恢复频道
- 登录、切图或 `/reload` 后，会自动加入配置里的频道（默认 `世界频道`）。
- 加入采用延迟 + 有限次重试，避免登录瞬间失败或刷屏。

### 5.6 频道快速切换条（默认关闭）
1. 在设置里勾选 **「启用频道切换条」**。
2. 聊天框上方出现一排按钮：**说 / 小队 / 团队 / 公会 / 世界 / 密语**。
3. 点击某个按钮 → 打开魔兽**原生聊天输入框**并切到对应频道，**光标就位，你打字后按回车发送**（插件不代发，零风险）。
4. 不在队伍/团队/公会时，对应按钮自动置灰；"世界"按钮需要已加入世界频道。
5. 按钮条可拖动；战斗中开启/移动会延迟到脱战后执行。

### 5.7 镜像面板（默认关闭）
可选的独立消息面板，把消息按频道着色镜像显示，**不接管主聊天框**。在设置里勾选「启用镜像面板」开启，可拖动。

## 6. 常见问题（FAQ）

**Q：加载后没反应 / 聊天框没提示加载信息？**
- 确认目录为 `AddOns\MyChat\MyChat.toc`（没有多套一层文件夹）。
- 角色选择界面的插件列表里确认 MyChat 已勾选。
- 若列表里根本没有 MyChat，多半是路径放错了版本目录（`_retail_` vs `_ptr_`）。

**Q：插件显示"已过期"无法加载？**
- 在插件列表勾选「加载过期插件」；或把 `MyChat.toc` 首行 `## Interface:` 改为你客户端的实际版本号。

**Q：频道切换条不显示？**
- 默认是关闭的，需在 `/mychat` 设置里勾选「启用频道切换条」。

**Q：某个功能出问题 / 聊天异常？**
- 先 `/mychat safe` 进入安全模式（只保留只读增强）。
- 再 `/mychat debug` 查看诊断信息。
- 仍有问题可 `/mychat reset` 恢复默认设置后 `/reload`。
- 最后可在插件列表取消勾选 MyChat，原生聊天不受任何影响。

**Q：设置会保存吗？**
- 会。设置存于 `SavedVariables\MyChatDB`，`/reload` 或重登后保留。`/mychat reset` 只重置功能开关，不清除最近密语/已加入频道等运行时数据。

---

## 7. 卸载

- 临时停用：角色选择界面的插件列表取消勾选 MyChat。
- 彻底删除：删除 `Interface\AddOns\MyChat\` 文件夹。存档 `WTF\...\SavedVariables\MyChatDB.lua` 可一并删除以清除配置。
